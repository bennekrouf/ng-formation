ng-formation
============

Exo de Formation

## Installation

	clone https://github.com/bennekrouf/ng-formation.git
	npm install

## Exécution

### Démarrer le serveur :

	node server.js

### Démarrer Selenium

	webdriver-manager start

### Démarrer le watcher de test :

	grunt test

### Lancer les tests fonctionnels :
	
	grunt
